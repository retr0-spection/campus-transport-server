Auth API Microservice Routes

POST /api/v1/auth/register

Registers a new user by creating an account.

Example Request Body: {
"email": "johndoe@example.com",
"password": "password123"
}
Example Response (201 Created): {
"message": "User registered successfully",
"user": {
"username": "johndoe",
"email": "johndoe@example.com"
}
}

POST /api/v1/auth/google-auth

Registers/logs in a user using Google.

Example Request Header: X-Google-Token: <idToken>
Example Response (200 OK): {
"message": "Authentication successful",
"token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
"refresh_token": "dGhpcyBpcyBhIHJlZnJlc2ggdG9rZW4..."
}

POST /api/v1/auth/login

Logs in a user and returns an authentication token.

Example Request Body: {
"email": "johndoe@example.com",
"password": "password123"
}
Example Response (200 OK): {
"message": "Login successful",
"token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
"refresh_token": "dGhpcyBpcyBhIHJlZnJlc2ggdG9rZW4..."
}

POST /api/v1/auth/logout

Logs out the currently authenticated user by invalidating their token.

Example Request Header: Authorization: Bearer <token>
Example Response (200 OK): {
"message": "Logout successful"
}

GET /api/v1/auth/me

Retrieves the currently authenticated user's details.

Example Request Header: Authorization: Bearer <token>
Example Response (200 OK): {
"id": "123456",
"email": "johndoe@example.com"
}

POST /api/v1/auth/refresh

Refreshes the authentication token.

Example Request Header: Authorization: Bearer <refresh_token>
Example Response (200 OK): {
"token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
"refresh_token": "dGhpcyBpcyBhIHJlZnJlc2ggdG9rZW4..."
}
